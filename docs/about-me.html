Who am I?

My name is Timothy Makoni. I'm a second-year Computer Science student at Swansea University and an aspiring software developer. Coming into higher education, I wasn't sure if CS was for me, but after my first year, I fell in love with the subject. If I'm not working on a project or learning a new technology, I'm probably browsing the latest news on tech updates. Outside of coding, I enjoy reading and playing chess.

About my Work...

My first year of university was mainly focused on learning to code in Java. In my second semester, we touched more on Object Oriented Programming and data structures such as Linked Lists, Stacks, Queues and Binary Trees. Again in the second semester, we had a module called 'Introduction to Data Science' in which we covered the basics of Jupyter Notebooks and the related software used in the data science process.
